Глава 1: Вводный урок

Фронтенд - это часть веб-разработки, которая отвечает за создание пользовательского интерфейса веб-сайта. Он включает в себя HTML, CSS и JavaScript.

Для того чтобы стать JUNIOR Фронтенд Разработчиком, вам нужно изучить основы HTML, CSS и JavaScript.

Смотрите видеоурок здесь для подробной информации.
